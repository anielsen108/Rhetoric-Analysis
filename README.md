# Rhetoric Analysis: 100 Famous Literary Passages

## Overview

This repository contains comprehensive rhetorical and syntactic analyses of 100 famous passages from world literature, spanning from ancient epic to contemporary critical theory. Each analysis examines how language creates meaning through both classical rhetorical devices and modern syntactic structures.

## Methodology

Each analysis combines two complementary approaches:

### 1. Classical Rhetoric
Identification and analysis of Greek, Roman, and modern rhetorical tropes including:
- Anaphora, epistrophe, chiasmus, antithesis
- Metaphor, simile, personification, hyperbole
- Litotes, zeugma, polysyndeton, asyndeton
- All terms include pronunciation guides (e.g., uh-NAF-or-uh) and effect analysis

### 2. Syntax as Style
Following Virginia Tufte's *Artful Sentences* methodology, analyzing:
- Sentence patterns (periodic, cumulative, loose)
- Modification strategies and word order
- Voice (active, passive), tense, and aspect
- Coordination vs. subordination
- Syntactic compression and expansion
- How grammatical choices create rhetorical effects

## Structure

Each analysis file includes:
1. **Author and Source** - Full attribution and publication information
2. **Passage** - Complete text of the excerpt analyzed
3. **Rhetorical Tropes** - 3-5 major devices with pronunciation, definition, example, and effect
4. **Syntax as Style Analysis** - Detailed examination of grammatical structures and their rhetorical functions
