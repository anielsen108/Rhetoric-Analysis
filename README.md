# Rhetoric Analysis Project

## Overview
Comprehensive rhetorical and syntactic analyses of 100 famous passages from world literature.

## Methodology
Each analysis combines:
1. **Classical Rhetoric**: Identification of Greek, Roman, and modern rhetorical tropes
2. **Syntax as Style**: Analysis following Virginia Tufte's *Artful Sentences* methodology
3. **Pronunciation Guides**: For all rhetorical terms
4. **Effect Analysis**: Explanation of how each device functions

## Structure
Each analysis includes:
- Author and source information
- Complete passage text
- Rhetorical tropes (with pronunciation and effect)
- Comprehensive syntax as style analysis
