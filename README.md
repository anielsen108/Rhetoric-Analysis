# Rhetoric Analysis: Literary Passages Organized by Decade

## Overview

This repository contains comprehensive rhetorical and syntactic analyses of literary passages from world literature, organized chronologically by decade. The collection spans from ancient epic to contemporary fiction, with particular depth in the period 1850-present. Each analysis examines how language creates meaning through both classical rhetorical devices and modern syntactic structures.

**Total Analyses: 125** (and growing)

## Methodology

Each analysis combines two complementary approaches:

### 1. Classical Rhetoric
Identification and analysis of Greek, Roman, and modern rhetorical tropes including:
- Anaphora, epistrophe, chiasmus, antithesis
- Metaphor, simile, personification, hyperbole
- Litotes, zeugma, polysyndeton, asyndeton
- All terms include pronunciation guides (e.g., uh-NAF-or-uh) and effect analysis

### 2. Syntax as Style
Following Virginia Tufte's *Artful Sentences* methodology, analyzing:
- Sentence patterns (periodic, cumulative, loose)
- Modification strategies and word order
- Voice (active, passive), tense, and aspect
- Coordination vs. subordination
- Syntactic compression and expansion
- How grammatical choices create rhetorical effects

## Structure

Each analysis file includes:
1. **Author and Source** - Full attribution and publication information
2. **Passage** - Complete text of the excerpt analyzed
3. **Rhetorical Tropes** - 3-5 major devices with pronunciation, definition, example, and effect
4. **Syntax as Style Analysis** - Detailed examination of grammatical structures and their rhetorical functions

## Usage

Each markdown file can be read independently. The analyses are designed to:
- Help students understand rhetorical devices in context
- Demonstrate Virginia Tufte's syntactic analysis methodology
- Show how grammar creates meaning across literary traditions
- Provide pronunciation guides for rhetorical terminology

## Project Goals

This is an ongoing project with the goal of achieving **20 analyses per decade** for the period 1850-present. Current coverage varies by decade, with some decades well-represented and others still being developed.
