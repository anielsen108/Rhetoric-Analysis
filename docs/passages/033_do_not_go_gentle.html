Syntax

Negative Imperative

"Do not go gentle"—the syntax uses negative command with unusual adverb. "Gentle" normally modifies verbs ("go gently"), but Thomas uses the adjective form, creating compressed, intense syntax. The command forbids passive acceptance.

Metaphoric ObjectMET-uh-for / ˈmɛtəfɔːr

"into that good night"—the prepositional phrase uses "that" (demonstrative) as if pointing to a known entity. The adjective "good" is bitterly ironic—death isn't good, but euphemism calls it so.

Modal Imperative

"Old age should burn and rave"—the modal auxiliary "should" expresses moral obligation. The two verbs "burn and rave" are violent, passionate—the syntax demands intensity from those usually expected to be calm.

Temporal EuphemismYOO-fuh-miz-um / ˈjuːfəmɪzəm

"at close of day"—another metaphoric prepositional phrase using time/light for life. The syntax maintains the extended metaphor: day = life, night = death.

Repeated Imperative

"Rage, rage against the dying of the light"—the epizeuxis (immediate repetition) creates drumbeat effect. The syntax is pure command, no modifiers until the prepositional phrase. The gerund "dying" makes death an ongoing process that can be resisted.
