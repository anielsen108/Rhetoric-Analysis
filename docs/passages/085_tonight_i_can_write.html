Stylistic Effect

Through modal transformation (cannot write → can write), imperative self-address, and nested quotation, Neruda creates syntax of grief becoming poetry—grammar that performs the very composition it describes.

Information flow: The opening establishes “Tonight I can write the saddest lines.”; subsequent material converts that initial footing into the new emphasis carried by “Write, for example, 'The night is starry and the stars are blue and shiver in the….”
