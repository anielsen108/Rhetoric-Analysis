# Premium analysis specification

An analysis is Premium only when it works as a complete reader dossier. The
section marker is necessary for the build, but it is not the quality standard.

## Required source structure

- A parseable passage title, author, work, year, and locus.
- Occasion/pressure, persona/stance, and a thesis of effect.
- A line-numbered primary-source passage.
- Separate sections for rhetorical tropes, rhetorical schemes, and modern
  syntactical approaches.
- Exact passage evidence and line references for every device.
- Definitions and effects for every trope and scheme, plus a near-miss or risk.
- Signals, stance effects, and classical kinships for syntax entries.
- Twelve full-dossier sections: prosody; syntax; deixis, aspect, and modality;
  image system; narrative mechanics; appeals; lineage; hotspots and
  faultlines; revision studio; imitatio; takeaways; and glossary.

## Corpus quality gates

- Every source analysis parses successfully.
- Every analysis contains at least one trope, one scheme, and one syntax entry.
- Every device names exact evidence that can be anchored to the passage.
- Every analysis contains all twelve numbered dossier sections.
- Generated HTML page count equals source analysis count.
- Existing hand-built Premium dossiers are never overwritten by the upgrader.

The upgrader preserves the compact analyses' primary text and critical claims.
It normalizes their structure, computes exact line references, and develops the
supporting dossier from each passage's own wording and existing observations.
